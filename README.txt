
***************************************************************************************
This package contains an implementation of the fusion algorithm described in
the paper:

"Quadtree-based multi-focus image fusion using a weighted focus-measure, Information Fusion 2015"

by Xiangzhi Bai, Yu Zhang, Fugen Zhou, and Bindang Xue.

Usage of this code is free for research purposes only. Please refer to the 
above publication if you use this code. 

This demo is implemented by Yu Zhang (zhangyu_buaa@126.com).

***************************************************************************************

Usage:

saliency_quad_tree.m - Quadtree-based multi-focus image fusion implementation.
